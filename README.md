# Менеджер расходов

**Автор:** Алексей Смирнов (пример)  
**Описание:** Веб-приложение для учёта личных расходов. Позволяет добавлять траты с суммой, категорией и датой, фильтровать по категории и датам, считать сумму за выбранный период, сохранять/загружать данные в JSON. Все данные автоматически сохраняются в localStorage браузера.

## Как использовать
1. Откройте `index.html` в любом современном браузере.
2. Заполните поля «Сумма» (только положительное число), «Категория», «Дата» и нажмите «Добавить расход».
3. Используйте фильтры по категории и диапазону дат. Сумма за период обновится автоматически.
4. Кнопки «Сохранить в JSON» выгружают текущие данные в файл, «Загрузить из JSON» – импортируют ранее сохранённые траты.
5. Данные не пропадут после перезагрузки страницы (localStorage).

## Пример тестов
- Добавить расход 200 руб., категория «Еда», дата 2026-04-22 → появится в таблице.
- Выбрать фильтр «Еда» → видна только эта запись.
- Задать период с 2026-04-01 по 2026-04-30 → сумма пересчитается.
- Попытка добавить отрицательную сумму → ошибка валидации.
- Сохранить JSON → скачается файл, после очистки можно загрузить обратно.
